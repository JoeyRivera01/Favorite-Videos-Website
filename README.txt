Title: Udacity Fresh Tomatoes Program

Required: You must be able to run Python 2.7 code.
Below is a link to the latest Python 2 release as of 11/10/2017.
https://www.python.org/downloads/release/python-2714/

Description: This program dynamically generates movie or TV show 
instances, creates an HTML file, opens your web browser, 
and displays the instances.

Steps to run the program:
1. Double click on entertainment_center.py or run from a CLI.
2. Note that an HTML document file is created in the current directory.
3. Note the browser window that opened and displayed TV or movie instances.

Optional:
In order to see the Movie instances, follow these steps:
1. Open entertainment_center.py in a python text editor.
2. Remove the comment from line 131, #fresh_tomatoes.open_movies_page(movies).
3. Comment out line 134, fresh_tomatoes.open_tv_page(tv_shows).
4. Save your changes.
5. Follow steps to run the program above.
